# Meu Portfólio 

Repositório do meu portfólio pessoal! Foi desenvolvido para apresentar minha trajetória na área de tecnologia, compartilhar um pouco sobre mim e destacar os projetos que venho construindo, como o *Fincanças Lite*.

🔗 **[Acesse o portfólio online aqui](https://link-do-seu-site.com)**

---

## Tecnologias Utilizadas

Este projeto foi construído vizando o aprendizado, entao utilizei o framework que venho aprendendo na ultima semana:

- **React.js** - Biblioteca principal para construção da interface.
- **CSS3 (Puro)** - Utilizando Flexbox, CSS Grid e animações (`@keyframes`).
- **Phosphor Icons** - Biblioteca de ícones minimalistas.

---

## Funcionalidades e Destaques do Código

- **Efeito de Digitação (Typewriter):** Animação CSS nativa feita com `steps()` para a apresentação inicial.
- **Renderização Dinâmica de Dados:** A seção de "Experiências" utiliza a separação de conceitos (dados vs. layout), renderizando abas interativas a partir de um arquivo `utils/experiences.js` isolado.
- **Design Responsivo:** Layout fluido que se adapta perfeitamente a dispositivos móveis (Mobile First).
